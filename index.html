O QUE É HTML?

HTML (Hyper Text Markup Language) é uma linguagem de marcação de texto.Ele usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da Web.

TAGS

São usadas para informar ao navegador a estrutura do site.Um elemento HTML é separado de outro texto ou em um documento por ¨tags¨, que consistem no nome do elemento entre ¨<¨¨>¨

TAGS MAIS USADAS

  • HTML- Define uma página html
  • HEAD- Cabeça
  • BODY- Corpo
  • TITLE- Titulo da pagina
  • P- Paragrafo
  • IMG - Inserir imagem
  • A- Links
